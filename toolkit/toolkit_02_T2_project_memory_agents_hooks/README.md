# Toolkit T2: Project memory, agents, and hooks

A deeper module on operational practices for AI-assisted research. Author the operational furniture that makes agentic research-coding sustainable for real projects: a project-memory `CLAUDE.md`, custom skills (e.g. an econometrics or backtest-validation skill), subagents for review and verification, and hooks that automate routine checks.

`cpu-light` · `standard` · best done after [Lecture 12 (B11)](../../lectures/lecture_12_B11_continuous_time_ha_numerics/README.md), before the surrogate / GP block

## Materials

- [`templates/CLAUDE_md_template.md`](templates/CLAUDE_md_template.md), project-memory starter.
- [`example_skill/SKILL.md`](example_skill/SKILL.md) and [`example_skill_strategic_revision/SKILL.md`](example_skill_strategic_revision/SKILL.md), skill templates.
- [`example_subagent/`](example_subagent/), subagent templates (econometrics-reviewer, monte-carlo-designer, backtest-validator, code-reviewer, doc-generator, test-writer, verifier).
- [`example_hooks/settings.json`](example_hooks/settings.json), hooks configuration sample.
- The exercise handout that covers both T1 and T2 lives in [`Toolkit T3`](../toolkit_03_T3_agentic_programming_exercises/README.md).
