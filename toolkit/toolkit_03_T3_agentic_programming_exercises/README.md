# Toolkit 03 (T3): Agentic programming, exercise handout

Self-paced exercises that accompany Toolkits T1 (the agentic research-coding
loop) and T2 (project memory, agents, and hooks). Twelve exercises in
total, seven workshop and five self-study, with solution sketches in the
T1 solutions folder.

## Learning goal

Work through the agentic-programming exercise handout that accompanies
T1 and T2, both the workshop set and the self-study extensions, and
consolidate the techniques into a personal research-coding playbook.

## When to do this

- **Recommended placement:** after Toolkit T2, as workshop and self-study practice.
- **Standalone:** can also be done independently after any AI-coding course.
