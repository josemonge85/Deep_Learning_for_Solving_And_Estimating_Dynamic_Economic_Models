# Readings, Lecture 04: IRBC with DEQNs

Default policy: link only.

## DEQN application

- **Azinovic, Gaegauf, Scheidegger (2022), "Deep Equilibrium Nets,"** *International Economic Review*, 63(4). [DOI](https://doi.org/10.1111/iere.12575). The IRBC application is the headline experiment of this paper.

## Comparison with related deep methods

- **Maliar, Maliar, Winant (2021), "Deep learning for solving dynamic economic models,"** *Journal of Monetary Economics*, 122. [DOI](https://doi.org/10.1016/j.jmoneco.2021.07.004). An alternative deep-learning route to large-scale RBC-style models.
- **Valaitis, Villa (2024), "A machine learning projection method for macro-finance models,"** *Quantitative Economics*. A projection-style alternative useful for benchmarking DEQNs.

## Classical IRBC reference

- **Kollmann, Maliar, Malin, Pichler (2011), "Comparison of solutions to the multi-country real business cycle model,"** *Journal of Economic Dynamics and Control*, 35(2). The reference comparison that DEQN-IRBC is benchmarked against.
